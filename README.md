## How to Install

- Clone repository
```cmd
git clone https://github.com/alfimonth/administrasi-perpustakaan.git
```

- Masuk ke folder `administrasi-perpustakaan`
```cmd
cd administrasi-perpustakaan
```

- Run app
```cmd
python main.py
```
